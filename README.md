# Econometrics-Group-Project-Analysing-Inequalities 
My econometrics university group project analyzing the primary factors affecting wage income—specifically education, experience, and race—using data from the 1940 U.S. Census. The project utilizes multiple Ordinary Least Squares (OLS) regression models and a Linear Probability Model (LPM) to explore income inequality and the varying returns to education
Authors, Adam Chaudhry, Mustafa Hayat, Mujtaba Sheikh, Max Vest 
We were graded a first class (1:1) on this project
